# Tmux 常用命令速查表

Tmux 是一个终端复用工具，允许你在一个终端窗口中同时运行多个会话，并支持分屏、后台运行。

## 🔑 核心概念与前缀键
*   **层级结构**：Session (会话) > Window (窗口/标签页) > Pane (窗格/分屏)
*   **前缀键 (Prefix)**：默认是 **`Ctrl` + `b`**。
    > **注意**：除 Shell 命令外，所有快捷键操作都需要先按下前缀键松开后，再按具体的指令键。

---

## 1. 命令行操作 (在 Shell 中执行)
这些命令是在**未进入 Tmux** 或**终端命令行**下输入的。

| 命令 | 说明 |
| :--- | :--- |
| `tmux` | 新建一个无名会话 |
| `tmux new -s <name>` | 新建一个名为 `<name>` 的会话 (**推荐**) |
| `tmux ls` | 查看当前所有运行中的会话 |
| `tmux a` | 恢复(Attach)上一次的会话 |
| `tmux a -t <name>` | 恢复指定名称的会话 |
| `tmux kill-session -t <name>` | 彻底杀死/关闭指定会话 |

---

## 2. 快捷键操作 (在 Tmux 内部)
**⚠️ 前置动作**：先按 `Ctrl` + `b`，松开后...

### 🖥️ 会话管理 (Session)
| 按键 | 说明 |
| :--- | :--- |
| **`d`** | **挂起(Detach)**：离开当前会话（程序在后台继续跑） |
| **`s`** | 交互式列表：查看并切换所有会话 |
| **`$`** | 重命名当前会话 |

### 🪟 窗格操作 (Pane - 分屏)
| 按键 | 说明 |
| :--- | :--- |
| **`%`** | **左右分屏** (Shift + 5) |
| **`"`** | **上下分屏** (Shift + ') |
| **`x`** | **关闭当前分屏** (下方提示按 y 确认) |
| **`z`** | **最大化/恢复**当前分屏 (Zoom，非常实用) |
| `方向键` | 在不同分屏间切换光标 |
| `Ctrl`+`方向键` | 调整当前分屏的大小 (按住 Ctrl 不放调整) |
| `SPACE` | 切换预置的分屏布局 |

### 📑 窗口操作 (Window - 标签页)
| 按键 | 说明 |
| :--- | :--- |
| **`c`** | **新建窗口** (Create) |
| **`w`** | **窗口列表**：可视化选择并切换窗口 |
| **`,`** | 重命名当前窗口 |
| **`n`** / **`p`** | 切换到 下一个(Next) / 上一个(Previous) 窗口 |
| `0`~`9` | 直接切换到指定编号的窗口 |

### 📜 翻页与查看模式
| 按键 | 说明 |
| :--- | :--- |
| **`[`** | **进入复制/翻页模式** (此时可用鼠标滚轮或 PgUp/PgDn) |
| **`q`** | **退出**复制/翻页模式 |
